# StylePro AI — Премиальный Барбершоп с ИИ-стилистом

Этот проект представляет собой современное веб-приложение для барбершопа, интегрирующее функции искусственного интеллекта для виртуальной примерки причесок.

## Основные возможности

- **ИИ-Визуализатор**: Загрузите фото и посмотрите, как на вас будут смотреться разные стрижки (используется AILabTools API).
- **Онлайн-запись**: Интуитивно понятный процесс выбора услуги, мастера и времени.
- **Админ-панель**: Управление записями, клиентами и услугами.
- **Премиальный дизайн**: Темная тема, современные шрифты (Space Grotesk) и плавная анимация.

## Технологический стек

- **Frontend**: Next.js 15 (App Router), React 19, Tailwind CSS.
- **UI Компоненты**: Shadcn UI, Lucide Icons.
- **ИИ**: Genkit + AILabTools API.
- **Backend**: Firebase App Hosting / Cloud Functions.

## Настройка для локальной разработки

1. **Клонируйте репозиторий**:
   ```bash
   git clone <url-вашего-репозитория>
   cd stylepro-ai
   ```

2. **Установите зависимости**:
   ```bash
   npm install
   ```

3. **Настройте переменные окружения**:
   Создайте файл `.env` в корне проекта и добавьте ваш API ключ:
   ```env
   AILAB_API_KEY=ваш_ключ_здесь
   ```

4. **Запустите проект**:
   ```bash
   npm run dev
   ```
   Откройте [http://localhost:9002](http://localhost:9002) в браузере.

## Публикация (Deployment)

Проект оптимизирован для развертывания на **Firebase App Hosting**:

1. Загрузите код в ваш репозиторий GitHub.
2. В консоли Firebase создайте новый проект **App Hosting**.
3. Подключите ваш репозиторий.
4. В разделе переменных окружения (Environment Variables) добавьте секрет `AILAB_API_KEY`.
5. Firebase автоматически соберет и опубликует ваш сайт.
