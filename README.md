# BarBerTok — Премиальный Барбершоп с ИИ-стилистом

Современное веб-приложение для барбершопа, интегрирующее функции искусственного интеллекта для виртуальной примерки причесок и полноценную панель управления.

## Основные возможности

- **ИИ-Визуализатор**: Загрузите фото и посмотрите, как на вас будут смотреться разные стрижки (используется AILabTools API).
- **Онлайн-запись**: Пошаговый процесс выбора услуги, мастера и времени с сохранением в Firestore.
- **Админ-панель**: Управление записями и статистикой (доступ по паролю).
- **Премиальный дизайн**: Темная тема, современные шрифты и плавная анимация.

## Технологический стек

- **Frontend**: Next.js 15 (App Router), React 19, Tailwind CSS.
- **Backend**: Firebase (Auth, Firestore, App Hosting).
- **ИИ**: Genkit + AILabTools API.

## Настройка доступа администратора

Данные для входа в админку (`/admin/login`) не хранятся в коде. Чтобы создать аккаунт:

1. Перейдите в [Firebase Console](https://console.firebase.google.com/).
2. Выберите ваш проект.
3. Перейдите в раздел **Authentication** -> **Users**.
4. Нажмите **Add user** и введите Email и Пароль.
5. Используйте эти данные для входа в панель управления сайтом.

## Развертывание (GitHub + Firebase)

1. **Создайте репозиторий на GitHub** и загрузите туда код.
2. **В консоли Firebase** выберите **App Hosting**.
3. **Подключите ваш GitHub репозиторий**.
4. **Environment Variables**: В настройках App Hosting добавьте секреты:
   - `AILAB_API_KEY`: Ваш ключ от ailabtools.com.
   - `NEXT_PUBLIC_FIREBASE_API_KEY`: Ваш API ключ Firebase.
   - `NEXT_PUBLIC_FIREBASE_PROJECT_ID`: ID вашего проекта.

## Локальная разработка

1. `npm install`
2. Создайте `.env.local` и добавьте туда ключи из раздела "Project Settings" вашего Firebase проекта.
3. `npm run dev`
