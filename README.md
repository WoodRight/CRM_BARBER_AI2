# BarBerTok AI — Премиальный Барбершоп с ИИ-стилистом

Современное веб-приложение для барбершопа, интегрирующее функции искусственного интеллекта для виртуальной примерки причесок.

## Основные возможности

- **ИИ-Визуализатор**: Загрузите фото и посмотрите, как на вас будут смотреться разные стрижки (используется AILabTools API с асинхронной обработкой).
- **Онлайн-запись**: Пошаговый процесс выбора услуги, мастера и времени.
- **Админ-панель**: Управление записями, клиентами и услугами.
- **Премиальный дизайн**: Темная тема, современные шрифты и плавная анимация.

## Технологический стек

- **Frontend**: Next.js 15 (App Router), React 19, Tailwind CSS.
- **UI Компоненты**: Shadcn UI, Lucide Icons.
- **ИИ**: Genkit + AILabTools API.
- **Backend**: Firebase App Hosting.

## Настройка для публикации (GitHub + Firebase)

1. **Создайте репозиторий на GitHub** и загрузите туда весь код.
2. **В консоли Firebase** создайте проект и выберите **App Hosting**.
3. **Подключите ваш GitHub репозиторий**.
4. **Важно:** В разделе **Environment Variables** (Переменные окружения) добавьте секрет:
   - `AILAB_API_KEY` = Ваш API ключ от ailabtools.com.
5. Firebase автоматически соберет проект и выдаст публичную ссылку.

## Локальная разработка

1. `npm install`
2. Создайте `.env` и добавьте `AILAB_API_KEY`.
3. `npm run dev`
